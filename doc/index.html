Module assets_locale

Translate textures and audios into different locale.

Info:

  • Author: 1F616EMO

Registration functions

register_asset (name, def) Register a translated asset
unregister_asset (name) Unregister a translated asset


Registration functions

register_asset (name, def)
Register a translated asset

Parameters:

  • name string The file name to be used in codes like node definition
  • def {[string]=string,...} Locale name as key, on-disk filename as value.

Usage:

    local MP = minetest.get_modpath("assets_locale")
assets_locale.register_asset("example_test.png",{
    en = MP .. "/textures/assets_locale_example.en.png",
    zh_TW = MP .. "/textures/assets_locale_example.zh_TW.png"
})
unregister_asset (name)
Unregister a translated asset

Parameters:

  • name string The file name to be used in codes like node definition

Usage:

    assets_locale.unregister_asset("example_test.png")
